🚀 Installation​

🌐 Production Deployment​

# 1. Clone source code
git clone https://github.com/vietvo371/AgriTrace.git
cd AgriTrace

# 2. Backend Setup
cd backend
cp .env.example .env
# ✏️ Fill in Database, Redis, JWT secret

# 3. Database Setup
mysql -u root -p
CREATE DATABASE agritrace;
CREATE USER 'agritrace_user'@'localhost' IDENTIFIED BY 'your_password';
GRANT ALL PRIVILEGES ON agritrace.* TO 'agritrace_user'@'localhost';
FLUSH PRIVILEGES;

# 4. Install Dependencies
yarn install

# 5. Run Migrations
yarn migrate

# 6. Seed Database
yarn seed

# 7. Start Production Server
yarn start:prod

💻 Local Development​

Backend Setup​

# 1. Install dependencies
yarn install

# 2. Setup environment
cp .env.example .env
# Edit .env with your local settings

# 3. Database setup
yarn db:migrate
yarn db:seed

# 4. Start development server
yarn dev

Mobile App Setup​

# 1. Install Expo CLI
npm install -g @expo/cli

# 2. Navigate to mobile app
cd mobile

# 3. Install dependencies
yarn install

# 4. Start Expo development server
yarn start
# or
expo start

Web Dashboard Setup​

# 1. Navigate to web dashboard
cd web

# 2. Install dependencies
yarn install

# 3. Start development server
yarn start

  • Test admin functions

4. Database​

  • Check migrations: yarn db:migrate:status
  • Verify seeded data
  • Test connections

🐛 Troubleshooting Common Issues​

Node.js Issues​

# Node version conflicts
nvm use 18
# or
nvm install 18

# Yarn issues
yarn cache clean
yarn install --force

React Native Issues​

# Metro bundler issues
npx react-native start --reset-cache

# Expo issues
expo doctor
expo install --fix

# iOS simulator issues
npx react-native run-ios --simulator="iPhone 14"

Database Issues​

# Connection issues
mysql -u root -p -e "SHOW DATABASES;"

# Migration issues
yarn db:migrate:rollback
yarn db:migrate

# Seeding issues
yarn db:seed --force

QR Code Issues​

# Camera permissions
# iOS: Add NSCameraUsageDescription to Info.plist
# Android: Add camera permission to AndroidManifest.xml

# QR generation issues
yarn add qrcode
yarn add react-native-qrcode-svg

📱 Mobile App Development​

React Native Setup​

# Install React Native CLI
npm install -g react-native-cli

# Create new project
npx react-native init AgriTraceMobile

# Install essential dependencies
yarn add @react-navigation/native
yarn add @react-navigation/stack
yarn add @react-navigation/bottom-tabs
yarn add react-native-camera
yarn add react-native-qrcode-scanner
yarn add @react-native-async-storage/async-storage
yarn add axios
yarn add react-native-vector-icons

Expo Setup​

# Install Expo CLI
npm install -g @expo/cli

# Create new Expo project
npx create-expo-app AgriTraceMobile

# Install Expo dependencies
npx expo install expo-camera
npx expo install expo-barcode-scanner
npx expo install expo-file-system
npx expo install expo-image-picker
npx expo install expo-location

Key Mobile Features​

  • Offline Support: Local data storage
